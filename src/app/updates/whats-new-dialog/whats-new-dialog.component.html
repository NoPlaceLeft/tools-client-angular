The people group selector is finally here.

January 2020

Copy & Paste & Import Subtree Changed

Previously: pasting or importing a subtree would replace the selected node. This was causing issues with some users who would accidentally delete data. So we have changed the functionality of Paste & Import Subtree.

Import: You can now only import data as a child node of the selected node.

Paste: You can choose between pasting as a child or replacing the entire Node and all it's Descendants.

Deprecation Warning

Copy & Paste & Import will soon be deprecated, in place of being able to link gen-maps together.

Country Selector

You can now select the country that the Group is located in.

Enhanced Geo Location Selector

You can now simply type in a ZipCode or PinCode and the app will use the central Latitude and Longitude.

People Groups Picker (deprecated)

Unfortunately, we were not capturing people groups accurately.

The People Groups selector has been removed. We apologize for the inconvenience and the loss of data.

We will be coming out with a more accurate People Groups selector soon.

Having an accurate People Groups selector will allow us to contribute People Group Data to the People Groups Project API.

New Date Picker

We have added a new date picker that will allow you to select the specific Year and Month.

The Original Date field has been deprecated, because the date format was incorrect and prevented us from creating accurate statistics.

We ask that you go to each group, and select the proper date using the new date picker.

Your original date will still be viewable. Once you pick the appropriate date, please click the red X button to remove the old date.
